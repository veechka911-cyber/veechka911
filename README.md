# Офтальмо — амбулаторный справочник (PWA)

Офлайн-приложение для практикующего офтальмолога. Карточки диагнозов, препаратов, алгоритмы и шкалы. Устанавливается на Android одним тапом из Chrome.

## Быстрый старт

После публикации в GitHub Pages приложение будет доступно по адресу:

**https://veechka911-cyber.github.io/veechka911/**

### Установка на Android

1. Откройте URL в Chrome
2. Дождитесь, пока приложение загрузится (~1 MB; индикатор «офлайн ✓» в шапке)
3. Меню Chrome → **«Установить приложение»** (или кнопка «📲 Установить» внизу)
4. Иконка появится на главном экране — открывается как нативное приложение, работает без интернета

### Установка на iOS (Safari)

1. Откройте URL в Safari
2. Поделиться → «На экран Домой»
3. Иконка появится на экране

## Контент

- **45+ нозологий** с разбором по 8 секциям: описание / симптомы / диагностика / диф.диагноз / тактика / препараты / маршрутизация / МКБ-10
- **80+ препаратов** (ATC S01 + системные ЛС, релевантные для офтальмолога + ЛС с глазными побочными эффектами)
- **15 пошаговых алгоритмов** (неотложка, скрининги, выбор АБ)
- **15 шкал и норм** (стадии глаукомы, ДРП, ВМД, остроты по возрастам, OSDI, ROP и др.)
- **Полный МКБ-10 H00-H59** с переходом по коду
- **130+ сокращений** офтальмолога с расшифровкой

## Деплой на GitHub Pages

В репозитории `veechka911-cyber/veechka911`:

1. Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **claude/offline-ophthalmology-app-DaYoN** (после merge — `main`)
4. Folder: **/ (root)**
5. Save

Через 1–3 минуты сайт будет доступен по URL.

## Структура

```
/
├── index.html                  — оболочка SPA
├── styles.css                  — стили (mobile-first, тёмная тема)
├── app.js                      — клиентский роутер, поиск, рендер
├── manifest.webmanifest        — PWA-манифест
├── sw.js                       — service worker (cache-first)
├── icons/                      — PWA-иконки (192, 512, maskable)
└── data/
    ├── diagnoses.js            — нозологии
    ├── drugs.js                — препараты
    ├── algorithms.js           — пошаговые алгоритмы
    ├── scales.js               — шкалы и классификации
    ├── icd10.js                — справочник МКБ-10 H00-H59
    └── abbreviations.js        — глоссарий сокращений
```
