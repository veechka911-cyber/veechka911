# Проект: Калькулятор для пациентов

## Статус
Планирование

## Идеи
- Калькулятор КБЖУ (калории, белки, жиры, углеводы)
- Калькулятор ИМТ с расшифровкой
- Норма воды в день
- Расчёт суточной калорийности по формуле Миффлина-Сан Жеора

## Инструменты
- Lovable (встроить на сайт или отдельная страница)
- Vercel (если отдельный мини-апп)

## Заметки
- Использовать бесплатный API продуктов для РФ
- Тест: проверить пустые результаты поиска по базе
