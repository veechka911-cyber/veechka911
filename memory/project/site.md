# Проект: Сайт-визитка врача

## Статус
Планирование

## Инструменты
- Lovable (no-code конструктор)
- Timeweb Cloud DB (PostgreSQL, ФЗ-152)
- Timeweb CDN (доступность в РФ)

## Структура сайта
- Главная (hero, преимущества, кнопка записи)
- Услуги (диетология + офтальмология)
- Обо мне
- Блог / статьи для пациентов
- Контакты и онлайн-запись

## Заметки
- Домен подключить как можно раньше
- Скрыть бейдж Lovable после релиза
- Mobile First — трафик из Telegram
